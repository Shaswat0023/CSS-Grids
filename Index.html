What is CSS Grid?

A grid is a collection of horizontal and vertical lines creating a pattern against which we can line up our design elements. They help us to create layouts in which our elements wont jump around or change width as we move from page, providing greater consistency on our websites.

    1. CSS Grid Layout is a two-dimensional layout system for the web.
    2. It lets you lay content out in rows and columns.
    3. It has many features that make building complex layouts straight forward.
    4. A grid will typically have columns, rows and then gaps between each row and column. The gaps are commonly referred to as gutters.
